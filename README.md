# Simulador de Investimentos (Teste automatizado)

Este projeto foi desenvolvido utilizando JavaScript, Node.js e Cypress tendo como objetivo automatizar um aplicativo de simulação de investimentos utilizando dados fictícios.

## Pré-requisitos

Este projeto conta com o NPM versão 6.13.4 e o framework Cypress.

Você pode baixar o aplicativo por meio do link:
https://download.cypress.io/desktop
. Após o download, descompacte-o no local de sua preferência. 

## Execução via CMD

Para executa-lo via CMD basta abrir a pasta do projeto via CMD e inserir o seguinte comando:
```bash
npm run cy:run
```

## Reports

Os resultados obtidos via CMD podem ser conferidos por meio dos reports HTML gerados via Mochawesome dentro da pasta mochawesome-report.
Também é possível ver o vídeo da execução, salvo dentro da pasta cypress > videos.
